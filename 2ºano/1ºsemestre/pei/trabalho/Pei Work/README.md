# Wildfires – Trabalho Prático PEI

## Estrutura do Projeto
- API: Implementação da API REST (Express + MongoDB)
- XSD: Schemas XML usados para validação
- Raw Data: Dados originais fornecidos
- Pipelines: Aggregations MongoDB
- Collections: Estrutura final da base de dados
- Postman: Testes da API

## Execução
1. Entrar na pasta API
2. npm install
3. npm start
4. Swagger disponível em /api/docs
